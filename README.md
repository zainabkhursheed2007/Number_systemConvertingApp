# Number_systemConvertingApp
This project is the culmination of my semester work for Computer Organization and Assembly Language (COAL). It bridges the gap between low-level hardware logic and modern software design by evolving a core 8086 Assembly engine into a sleek C++/CLI Windows Form application.
The goal was to transform complex manual conversions into an interactive experience. I implemented a "Guess and Check" Challenge Mode that provides real-time feedback, making it an essential tool for students to practice and master number systems.
Languages: 8086 Assembly, C++/CLI
Tools: emu8086, Visual Studio
